## 1. 中考英语 · 目标 100+

**核心目标**：本周内完成至少 2 套中考英语模拟卷，单卷得分稳定站在 100 分以上。

**拆解动作**：
- 听力：每日精听 1 篇，抓关键词与转折信号
- 阅读：限时训练，错题归类（细节题 / 推理题 / 词义猜测）
- 写作：背熟 3 套通用模板（记叙文 / 议论文 / 书信），每周仿写 1 篇
- 复盘：建立错题本，周末回看本周高频失分点

> [!TIP]
> 100 分不是终点，而是「基础扎实」的底线。先保住底盘，再冲高分。

## 2. STM32 入门

**核心目标**：完成开发环境搭建，并跑通 2-3 个最基础的外设实验。

**拆解动作**：
- 环境：安装 STM32CubeIDE，配好下载器（ST-Link / DAP-Link）
- 实验一：LED 闪烁（GPIO 输出 + 延时）
- 实验二：按键输入（GPIO 输入 + 中断）
- 实验三：串口打印（USART 通信，用串口助手看输出）
- 概念：理解时钟树、寄存器 vs HAL 库的差异

> [!NOTE]
> 入门阶段别急着啃底层寄存器，先用 HAL 库把「点灯→按键→串口」这条主线走通，建立手感最重要。

## 3. 机器学习入门

**核心目标**：建立机器学习整体认知，并用 sklearn 跑通第一个端到端模型。

**拆解动作**：
- 概念：区分监督 / 无监督学习，理解回归与分类
- 工具：装好 Python + NumPy + Pandas + scikit-learn
- 实战：用经典数据集（如鸢尾花分类 / 波士顿房价回归）走完「数据 → 训练 → 评估」全流程
- 输出：写一篇学习笔记，讲清「模型到底在学什么」

> [!TIP]
> 机器学习最怕「只看公式不动手」。第一个模型不求漂亮，求跑通、求看懂每一行在干嘛。
