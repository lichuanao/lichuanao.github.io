这是博客的第一篇文章。🎉

这个站点基于 **Jekyll + Chirpy + GitHub Pages** 搭建，完全免费且极易维护：你只需要写 Markdown，剩下的构建与发布都由 GitHub 自动完成。

## 日常怎么写文章？

1. 在 `_posts/` 目录下新建一个文件，命名格式为 `YYYY-MM-DD-标题.md`
2. 在文件开头写一段 **front matter**（见下文）
3. 用 Markdown 写正文
4. `git add` / `commit` / `push` 到 GitHub，几分钟后文章自动上线

### 文章头信息(front matter)示例

```yaml
---
title: 文章标题
date: 2026-07-12 10:00:00 +0800
categories: [技术]
tags: [Jekyll, 教程]
---
```
